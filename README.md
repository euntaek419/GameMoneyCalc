# GameMoneyCalc

고려 중 사항
1. 상품권 옵션 설정 입력에 따른 동작
2. 현금 거래 비율 입력에 따른 동작
3. 캐시샵 판매 금액과 경매장 판매 금액의 비교 및 CSS 설정
4. % 할인과 % 추가증정의 동작 및 
5. 현금 거래 비율에서 REVERSE 버튼으로 비율의 주체를 변경
6. 상품권 옵션 설정에 따른 결과 설정
